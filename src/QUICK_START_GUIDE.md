## â“ TROUBLESHOOTING

### "I don't see the Messages tab"
- Make sure you selected the correct role
- Members and Officers both have Messages
- Network Admins see it in main navigation

### "Photos won't upload"
- Check file size (max 10MB per photo)
- Ensure file type is JPG, PNG, or GIF
- Try drag-and-drop instead of browse
- Check internet connection

### "Can't send a message"
- Type in message field first
- Press Enter or click Send button
- Check you're not blocked
- Ensure recipient exists

### "Dues payment failed"
- Check card details
- Ensure sufficient funds
- Try different payment method
- Contact officer for help
